# 이미지 압축기

이미지를 1.9MB 이하로 압축하는 간단한 웹 애플리케이션입니다. 해상도는 유지하면서 화질 손상을 최소화하여 압축합니다.

## 기능

- **드래그 앤 드롭**: 이미지를 드래그하여 업로드
- **파일 선택**: 클릭하여 이미지 파일 선택
- **자동 압축**: 1.9MB 이하로 자동 압축
- **해상도 유지**: 원본 이미지의 해상도를 그대로 유지
- **화질 최적화**: 품질을 자동으로 조정하여 최대한 화질 손상 최소화
- **미리보기**: 원본과 압축된 이미지를 비교하여 확인
- **다운로드**: 압축된 이미지를 한 번의 클릭으로 다운로드

## 제한사항

- **최대 업로드 용량**: 100MB
- **지원 형식**: 모든 이미지 형식 (JPEG, PNG, WebP 등)
- **압축 목표**: 1.9MB 이하

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 이미지를 드래그하여 업로드 영역에 놓거나, "파일 선택" 버튼을 클릭하여 이미지를 선택합니다.
3. 자동으로 압축이 시작되며 진행 상태가 표시됩니다.
4. 압축이 완료되면 원본과 압축된 이미지를 비교할 수 있습니다.
5. "다운로드" 버튼을 클릭하여 압축된 이미지를 저장합니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- [browser-image-compression](https://github.com/Donaldcwl/browser-image-compression) (CDN)

## 브라우저 호환성

모든 최신 브라우저에서 동작합니다:
- Chrome
- Firefox
- Safari
- Edge

## 주의사항

- 모든 처리는 브라우저에서 클라이언트 사이드로 수행되므로, 서버로 이미지가 전송되지 않습니다.
- 매우 큰 이미지(100MB에 가까운)의 경우 압축에 시간이 걸릴 수 있습니다.
- 브라우저의 메모리 제한으로 인해 일부 매우 큰 이미지는 처리하지 못할 수 있습니다.
